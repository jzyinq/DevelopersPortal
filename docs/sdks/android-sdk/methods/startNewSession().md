# startNewSession()

The **startNewSession()** method starts a new session.

## Syntax

<div class="tabs">

<div class="group-tab">

Java

``` javascript
getTracker().startNewSession();
```

</div>

<div class="group-tab">

Kotlin

``` javascript
tracker.startNewSession()
```

</div>

</div>
