# getDeviceId()

The **getDeviceId()** method returns a device ID set for a given user.

## Syntax

<div class="tabs">

<div class="group-tab">

Java

``` javascript
getTracker().getDeviceId();
```

</div>

<div class="group-tab">

Kotlin

``` javascript
tracker.deviceId
```

</div>

</div>

## Returns

The unique device ID  
Format: Example: abcd123e-a123-bcFG-d123  
Type: String
